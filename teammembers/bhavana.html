CLI Learns

I learned how to use the CLI to interact with Git and GitHub efficiently, performing tasks like commits, branches, and pushes through terminal commands.

Q: How do you switch between branches using the CLI?

A: You can switch between branches using the command git checkout branch-name. In newer Git versions, you can also use git switch branch-name for better readability.
